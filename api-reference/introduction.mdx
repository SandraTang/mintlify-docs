## Καλώς ήρθατε

Υπάρχουν δύο τρόποι για να δημιουργήσετε τεκμηρίωση API: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) και [MDX components](https://mintlify.com/docs/api-playground/mdx/configuration). Για το starter kit, χρησιμοποιούμε την ακόλουθη προδιαγραφή OpenAPI.

## Αυθεντικοποίηση

Όλα τα API endpoints αυθεντικοποιούνται χρησιμοποιώντας Bearer tokens και λαμβάνονται από το αρχείο προδιαγραφών.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
