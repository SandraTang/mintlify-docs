## Básico

### Código en Línea

Para denotar una `palabra` o `frase` como código, enciérrela entre comillas invertidas (`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### Bloque de Código

Use [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres comillas invertidas y coloque después de las comillas el lenguaje de programación de su fragmento para obtener resaltado de sintaxis. Opcionalmente, también puede escribir el nombre de su código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
