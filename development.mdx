<Info>
  **Προαπαιτούμενο**: Παρακαλώ εγκαταστήστε το Node.js (έκδοση 19 ή νεότερη) πριν συνεχίσετε.
</Info>

Ακολουθήστε αυτά τα βήματα για να εγκαταστήσετε και να εκτελέσετε το Mintlify στο λειτουργικό σας σύστημα:

**Βήμα 1**: Εγκατάσταση Mintlify:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**Βήμα 2**: Μεταβείτε στον κατάλογο εγγράφων (όπου βρίσκεται το αρχείο `mint.json`) και εκτελέστε την ακόλουθη εντολή:

```bash
mintlify dev
```

Μια τοπική προεπισκόπηση των εγγράφων σας θα είναι διαθέσιμη στη διεύθυνση `http://localhost:3000`.

### Προσαρμοσμένες Θύρες

Από προεπιλογή, το Mintlify χρησιμοποιεί τη θύρα 3000. Μπορείτε να προσαρμόσετε τη θύρα στην οποία εκτελείται το Mintlify χρησιμοποιώντας τη σημαία `--port`. Για να εκτελέσετε το Mintlify στη θύρα 3333, για παράδειγμα, χρησιμοποιήστε αυτή την εντολή:

```bash
mintlify dev --port 3333
```

Εάν προσπαθήσετε να εκτελέσετε το Mintlify σε μια θύρα που χρησιμοποιείται ήδη, θα χρησιμοποιήσει την επόμενη διαθέσιμη θύρα:

```md
Η θύρα 3000 χρησιμοποιείται ήδη. Δοκιμή της θύρας 3001.
```

## Εκδόσεις Mintlify

Παρακαλώ σημειώστε ότι κάθε έκδοση του CLI συνδέεται με μια συγκεκριμένη έκδοση του Mintlify. Εάν ο τοπικός σας ιστότοπος δεν ευθυγραμμίζεται με την έκδοση παραγωγής, παρακαλώ ενημερώστε το CLI:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Επικύρωση Συνδέσμων

Το CLI μπορεί να βοηθήσει στην επικύρωση των συνδέσμων αναφοράς στα έγγραφά σας. Για να εντοπίσετε τυχόν σπασμένους συνδέσμους, χρησιμοποιήστε την ακόλουθη εντολή:

```bash
mintlify broken-links
```
