<Info>
  **متطلب أساسي**: يرجى تثبيت Node.js (الإصدار 19 أو أعلى) قبل المتابعة.
</Info>

اتبع هذه الخطوات لتثبيت وتشغيل Mintlify على نظام التشغيل الخاص بك:

**الخطوة 1**: تثبيت Mintlify:

<CodeGroup>

  ```bash npm
  npm i -g mintlify
  ```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

**الخطوة 2**: انتقل إلى دليل المستندات (حيث يوجد ملف `mint.json`) وقم بتنفيذ الأمر التالي:

```bash
mintlify dev
```

ستكون المعاينة المحلية للوثائق الخاصة بك متاحة على `http://localhost:3000`.

### المنافذ المخصصة

افتراضياً، يستخدم Mintlify المنفذ 3000. يمكنك تخصيص المنفذ الذي يعمل عليه Mintlify باستخدام علامة `--port`. لتشغيل Mintlify على المنفذ 3333، على سبيل المثال، استخدم هذا الأمر:

```bash
mintlify dev --port 3333
```

إذا حاولت تشغيل Mintlify على منفذ قيد الاستخدام بالفعل، فسيستخدم المنفذ التالي المتاح:

```md
المنفذ 3000 قيد الاستخدام بالفعل. جارٍ محاولة استخدام المنفذ 3001 بدلاً من ذلك.
```

## إصدارات Mintlify

يرجى ملاحظة أن كل إصدار من واجهة سطر الأوامر (CLI) مرتبط بإصدار معين من Mintlify. إذا لم يتطابق موقعك المحلي مع نسخة الإنتاج، يرجى تحديث واجهة سطر الأوامر:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## التحقق من صحة الروابط

يمكن لواجهة سطر الأوامر المساعدة في التحقق من صحة روابط المراجع في وثائقك. للتعرف على أي روابط معطلة، استخدم الأمر التالي:

```bash
mintlify broken-links
```

## تنسيق الكود

نقترح استخدام إضافات على بيئة التطوير المتكاملة (IDE) الخاصة بك للتعرف على MDX وتنسيقه. إذا كنت تستخدم VSCode، فكر في استخدام [إضافة MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) لإبراز بناء الجملة، و[Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) لتنسيق الكود.

## استكشاف الأخطاء وإصلاحها

<AccordionGroup>
  <Accordion title='خطأ: تعذر تحميل وحدة "sharp" باستخدام وقت تشغيل darwin-arm64'>

    قد يكون هذا بسبب إصدار قديم من node. جرب ما يلي:
    1. قم بإزالة الإصدار المثبت حالياً من mintlify: `npm remove -g mintlify`
    2. قم بالترقية إلى Node v19 أو أعلى.
    3. أعد تثبيت mintlify: `npm install -g mintlify`
  </Accordion>

  <Accordion title="مشكلة: مواجهة خطأ غير معروف">
  
    الحل: انتقل إلى جذر جهازك واحذف المجلد ~/.mintlify. بعد ذلك، قم بتشغيل `mintlify dev` مرة أخرى.
  </Accordion>
</AccordionGroup>
