### Ενσωματωμένος Κώδικας

Για να επισημάνετε μια `λέξη` ή `φράση` ως κώδικα, περικλείστε την σε ανάποδα εισαγωγικά (`).

### Μπλοκ Κώδικα

Χρησιμοποιήστε [μπλοκ κώδικα με περίφραξη](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) περικλείοντας τον κώδικα σε τρία ανάποδα εισαγωγικά και ακολουθήστε τα αρχικά εισαγωγικά με τη γλώσσα προγραμματισμού του αποσπάσματός σας για να έχετε επισήμανση σύνταξης. Προαιρετικά, μπορείτε επίσης να γράψετε το όνομα του κώδικά σας μετά τη γλώσσα προγραμματισμού.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
