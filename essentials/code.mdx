## أساسيات

### الكود المضمن

لتمييز `كلمة` أو `عبارة` ككود برمجي، قم بإحاطتها بعلامات الاقتباس العكسية (`).

```
لتمييز `كلمة` أو `عبارة` ككود برمجي، قم بإحاطتها بعلامات الاقتباس العكسية (`).
```

### كتلة البرمجة

استخدم [كتل البرمجة المسيجة](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) عن طريق إحاطة الكود بثلاث علامات اقتباس عكسية واتبع العلامات الأولى بلغة البرمجة الخاصة بمقتطف الكود للحصول على تمييز بناء الجملة. اختيارياً، يمكنك أيضاً كتابة اسم الكود الخاص بك بعد لغة البرمجة.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
