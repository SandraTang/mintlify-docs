Κάθε ιστότοπος Mintlify χρειάζεται ένα αρχείο `mint.json` με τις βασικές ρυθμίσεις διαμόρφωσης. Μάθετε περισσότερα για τις [ιδιότητες](#properties) παρακάτω.

## Ιδιότητες

<ResponseField name="name" type="string" required>
Όνομα του έργου σας. Χρησιμοποιείται για τον γενικό τίτλο.

Παράδειγμα: `mintlify`

</ResponseField>

<ResponseField name="navigation" type="Navigation[]" required>
  Ένας πίνακας ομάδων με όλες τις σελίδες εντός αυτής της ομάδας
  <Expandable title="Πλοήγηση">
    <ResponseField name="group" type="string">
    Το όνομα της ομάδας.

    Παράδειγμα: `Ρυθμίσεις`

    </ResponseField>
    <ResponseField name="pages" type="string[]">
    Οι σχετικές διαδρομές προς τα αρχεία markdown που θα χρησιμεύσουν ως σελίδες.

    Παράδειγμα: `["customization", "page"]`

    </ResponseField>

  </Expandable>
</ResponseField>

<ResponseField name="logo" type="string or object">
  Διαδρομή προς την εικόνα του λογότυπου ή αντικείμενο με διαδρομή προς εικόνες λογότυπου για "φωτεινή" και "σκοτεινή" λειτουργία
  <Expandable title="Λογότυπο">
    <ResponseField name="light" type="string">
      Διαδρομή προς το λογότυπο σε φωτεινή λειτουργία
    </ResponseField>
    <ResponseField name="dark" type="string">
      Διαδρομή προς το λογότυπο σε σκοτεινή λειτουργία
    </ResponseField>
    <ResponseField name="href" type="string" default="/">
      Πού σας οδηγεί το κλικ στο λογότυπο
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="favicon" type="string">
  Διαδρομή προς την εικόνα favicon
</ResponseField>

<ResponseField name="colors" type="Colors">
  Κωδικοί χρωμάτων Hex για το γενικό θέμα σας
  <Expandable title="Χρώματα">
    <ResponseField name="primary" type="string" required>
      Το κύριο χρώμα. Χρησιμοποιείται συχνότερα για επισημασμένο περιεχόμενο, επικεφαλίδες ενοτήτων, τονισμούς, σε φωτεινή λειτουργία
    </ResponseField>
    <ResponseField name="light" type="string">
      Το κύριο χρώμα για σκοτεινή λειτουργία. Χρησιμοποιείται συχνότερα για επισημασμένο περιεχόμενο, επικεφαλίδες ενοτήτων, τονισμούς, σε σκοτεινή λειτουργία
    </ResponseField>
    <ResponseField name="dark" type="string">
      Το κύριο χρώμα για σημαντικά κουμπιά
    </ResponseField>
    <ResponseField name="background" type="object">
      Το χρώμα του φόντου τόσο σε φωτεινή όσο και σε σκοτεινή λειτουργία
      <Expandable title="Αντικείμενο">
        <ResponseField name="light" type="string" required>
          Ο κωδικός χρώματος hex του φόντου σε φωτεινή λειτουργία
        </ResponseField>
        <ResponseField name="dark" type="string" required>
          Ο κωδικός χρώματος hex του φόντου σε σκοτεινή λειτουργία
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="topbarLinks" type="TopbarLink[]">
  Πίνακας με `ονόματα` και `url` συνδέσμων που θέλετε να συμπεριλάβετε στην επάνω μπάρα
  <Expandable title="TopbarLink">
    <ResponseField name="name" type="string">
    Το όνομα του κουμπιού.

    Παράδειγμα: `Επικοινωνήστε μαζί μας`
    </ResponseField>
    <ResponseField name="url" type="string">
    Το url όταν κάνετε κλικ στο κουμπί. Παράδειγμα: `https://mintlify.com/docs`
    </ResponseField>

  </Expandable>
</ResponseField>
