قائمة التنقل هي قائمة الروابط الموجودة في كل موقع.

من المحتمل أن تقوم بتحديث `mint.json` في كل مرة تضيف فيها صفحة جديدة. الصفحات لا تظهر تلقائياً.

## صيغة التنقل

صيغة التنقل لدينا متكررة مما يعني أنه يمكنك إنشاء مجموعات تنقل متداخلة. لست بحاجة إلى تضمين `.mdx` في أسماء الصفحات.

<CodeGroup>

```json التنقل العادي
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json التنقل المتداخل
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

## المجلدات

ما عليك سوى وضع ملفات MDX في مجلدات وتحديث المسارات في `mint.json`.

على سبيل المثال، لإنشاء صفحة في `https://yoursite.com/your-folder/your-page` ستقوم بإنشاء مجلد يسمى `your-folder` يحتوي على ملف MDX يسمى `your-page.mdx`.

<Warning>

لا يمكنك استخدام `api` كاسم لمجلد إلا إذا قمت بتضمينه داخل مجلد آخر. يستخدم Mintlify إطار Next.js الذي يحتفظ بمجلد `api` على المستوى الأعلى لاستدعاءات الخادم الداخلية. سيتم قبول اسم مجلد مثل `api-reference`.

</Warning>

```json التنقل مع المجلد
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
]
```

## الصفحات المخفية

لن تظهر ملفات MDX غير المضمنة في `mint.json` في الشريط الجانبي ولكن يمكن الوصول إليها من خلال شريط البحث وعن طريق الربط المباشر بها.
