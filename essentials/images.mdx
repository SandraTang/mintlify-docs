## Εικόνα

### Χρήση Markdown

Η [σύνταξη markdown](https://www.markdownguide.org/basic-syntax/#images) σας επιτρέπει να προσθέτετε εικόνες χρησιμοποιώντας τον ακόλουθο κώδικα

```md
![title](/path/image.jpg)
```

Σημειώστε ότι το μέγεθος του αρχείου εικόνας πρέπει να είναι μικρότερο από 5MB. Διαφορετικά, συνιστούμε τη φιλοξενία σε μια υπηρεσία όπως το [Cloudinary](https://cloudinary.com/) ή το [S3](https://aws.amazon.com/s3/). Στη συνέχεια, μπορείτε να χρησιμοποιήσετε αυτό το URL και να το ενσωματώσετε.

### Χρήση Ενσωματώσεων

Για περισσότερη προσαρμοστικότητα με τις εικόνες, μπορείτε επίσης να χρησιμοποιήσετε [ενσωματώσεις](/writing-content/embed) για να προσθέσετε εικόνες

```html
<img height="200" src="/path/image.jpg" />
```

</Tip>

### iFrames

Φορτώνει μια άλλη σελίδα HTML μέσα στο έγγραφο. Χρησιμοποιείται συνήθως για την ενσωμάτωση βίντεο.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
